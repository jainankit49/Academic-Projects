Tools/IDE/Server Required

1.Microsoft Visual Studio 2012
2.NetBeans 7.4
3.Glassfish Server 3.1.2
4.Solr 4.5.1
5.Cygwin 


Setup tools

1.Install Microsoft Visual Studio 2012
2.Install Glassfish server 3.1.2
3.Install Netbeans 7.4
4.Add Glassfish server in Netbeans 7.4
5.Unzip solr zip file
6.Install Cygwin


Setup Project

1.Open Glue microsoft visual studio project in VS 2012.
2.Open test project in Netbeans 7.4 and deploy the web services.
3.Update service referrences in Glue VS project
4.Open Cygwin -> Goto {solr folder}/example -> run : java -jar start.jar

All set to run the QA System.


How to Run

1.Go to Glue project in VS 2012.
2.Build the project.
3.press F5 or goto Debug -> Start Debugging.
